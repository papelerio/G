Configuración para GitHub

1. Crea un repositorio en GitHub

2. Sube este archivo HTML como index.html

3. Crea un archivo llamado blog-data.json con contenido inicial: {"posts": [], "categories": []}

4. Activa GitHub Pages en la configuración del repositorio

5. Para actualizaciones automáticas, necesitarías usar GitHub API o GitHub Pages con un enfoque especial
